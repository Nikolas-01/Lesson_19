Общее описание:
1.1. Проект представляет собой динамический сайт, позволяющий осуществлять парсинг с сайта auto.drom.ru по моделям авто и выводит среднюю цену по России на автомобиль.
1.2. Создан на python с использованием фреймворка Flask.
Навигация:
2.1. Навигация сайта: Главная страница - Парсер - Контакты.
2.2. С Главной страницы можно передать из формы название модели авто, выводится на новой странице с url-слагом /cars_form.
2.3. На странице "Парсинг" осуществляется парсинг по модели авто.
Выводится название модели авто, название сайта, с которого парсится информация, высчитывается средняя цена, минимальная и максимальная, количество предложений.
2.4. На странице "Контакты" есть формы для ввода имени, е-мейла и форма для текста.
3.1. В бэк-энде проекта можно создать базу данных - в базу данных сохраняются данные парсинга: таблицы с названием автомобиля, годом автомобиля, ценой автомобиля.
3.2. Проект позволяет создать базу данных с помощью ORM sqlite или sqlAlchemy.
